## ğŸ“¦ Installation & Setup

### Option 1: Direct File Access (Simple)
1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/abc-digital-website.git
cd abc-digital-website
```

2. Start a local server (required for component loading)
```bash
# Using Python 3
python -m http.server 8000

# Or using Python 2
python -m SimpleHTTPServer 8000
```

3. Open browser and navigate to:
```
http://localhost:8000
```

### Option 2: Using the Provided Script (Windows)
1. Clone the repository
2. Double-click `START-WEBSITE.bat`
3. Browser will open automatically
